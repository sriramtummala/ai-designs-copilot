# Keyboard Navigation Guidelines
 
## Principle
 
All interactive elements on a web page must be accessible and operable via keyboard alone, without requiring a mouse.
 
## Requirements
 
1.  **Tab Order:** Interactive elements must be reachable in a logical and predictable order using the `Tab` key.
2.  **Focus Indicator:** A clear visual indicator must be present for the currently focused element.
3.  **Interactive Elements:** All buttons, links, form fields, and custom interactive components must be operable using `Enter`, `Space`, or arrow keys as appropriate.
4.  **Skip Links:** Provide a "Skip to main content" link for users to bypass repetitive navigation.
 
## Testing
 
*   Use the `Tab` key to navigate through the entire page.
*   Ensure all interactive elements can be activated.
*   Verify that no keyboard traps exist (where focus gets stuck).
