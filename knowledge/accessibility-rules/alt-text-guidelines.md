# Alt Text Guidelines
 
## Purpose
 
Alternative text (alt text) provides a textual description of images for users who cannot see them, including screen reader users and search engines.
 
## Best Practices
 
1.  **Be Concise:** Aim for 125 characters or less.
2.  **Be Descriptive:** Convey the essential information or function of the image.
3.  **Avoid Redundancy:** Do not start with "Image of..." or "Picture of...".
4.  **Context is Key:** The alt text should make sense in the context of the surrounding content.
5.  **Decorative Images:** For purely decorative images that convey no information, use an empty alt attribute (`alt=""`).
 
## Examples
 
| Image | Context | Good Alt Text | Bad Alt Text |
|---|---|---|---|
| Chart showing sales growth | In an article about Q3 earnings | "Bar chart showing 15% sales growth in Q3." | "Chart" |
| Magnifying glass icon | Next to a search input | "Search" | "Magnifying glass" |
| Abstract background pattern | On a landing page | `alt=""` | "Abstract pattern" |
