# Typography Scale Design Tokens
 
## Overview
 
Our typography scale ensures a consistent and hierarchical visual rhythm across all content. It defines font families, sizes, weights, and line heights.
 
## Font Families
 
*   `--font-family-base`: Primary font for body text (e.g., Inter, sans-serif).
*   `--font-family-heading`: Font for headings and display text (e.g., Montserrat, sans-serif).
 
## Font Sizes
 
*   `--font-size-xs`: Extra small text (e.g., 12px).
*   `--font-size-sm`: Small text (e.g., 14px).
*   `--font-size-md`: Medium text (e.g., 16px, default body).
*   `--font-size-lg`: Large text (e.g., 20px).
*   `--font-size-xl`: Extra large text (e.g., 24px).
 
## Font Weights
 
*   `--font-weight-regular`: 400
*   `--font-weight-medium`: 500
*   `--font-weight-bold`: 700
 
## Usage
 
Use semantic typography tokens to ensure consistency. For example, `font-size: var(--font-size-md);`.
