# Spacing System Design Tokens
 
## Overview
 
Our spacing system is based on a 4-pixel grid, providing a consistent rhythm for layout and component spacing. All spacing values are defined as multiples of 4px.
 
## Scale
 
*   `--spacing-xs`: 4px (extra small)
*   `--spacing-sm`: 8px (small)
*   `--spacing-md`: 16px (medium)
*   `--spacing-lg`: 24px (large)
*   `--spacing-xl`: 32px (extra large)
 
## Usage
 
Apply spacing tokens for margins, paddings, and gaps between elements. For example, `margin-bottom: var(--spacing-md);`.
