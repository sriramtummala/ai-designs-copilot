# Image Optimization for CMS
 
## Purpose
 
Optimizing images improves page load times, enhances user experience, and contributes to SEO.
 
## Guidelines
 
1.  **File Format:**
    *   **JPEG:** For photographs and complex images.
    *   **PNG:** For images with transparency or sharp edges (logos, icons).
    *   **WebP/AVIF:** Preferred modern formats for better compression and quality.
2.  **Compression:** Use image compression tools to reduce file size without significant loss of quality.
3.  **Dimensions:** Resize images to their display dimensions. Avoid uploading unnecessarily large images.
4.  **Responsive Images:** Use `srcset` and `sizes` attributes to serve different image sizes based on screen resolution.
5.  **Alt Text:** Always provide descriptive alt text for accessibility and SEO.
 
## Tools
 
*   Image CDN (e.g., Cloudinary, Imgix)
*   Image compression software (e.g., TinyPNG, Squoosh)
